# Bug Report v3.0.1 - 2026-01-14

## Bug #1: [Obmannzuweisung fehlerhaft]
**Priority:** CRITICAL
**Feature:** [Admin-Backend - Obmannzuweisung]

**Problem:**
Im Admin-Backend im Tab Obleute kann man zwar die angemeldeten Nutzer der Webseite auswählen, ebenfalls die Meldegruppen und Wildarten, aber eine Zuwesiung lässt sich nicht speichern. Die Zuweisung ist wichtig für die Unterscheidung in der ERfassung der Meldungen, denn die Vor-Auswahl der Meldegruppe hängt von dem angemeldeten Obmann ab.
