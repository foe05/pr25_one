Übersicht & CSV Export URLs

Export URLs

Verwenden Sie diese URLs für automatische CSV-Exporte:

Grundlegende Export URLs
{{ request.url_root }}export_csv
Exportiert alle Einträge
Filteroptionen
Parameter Beispiel Beschreibung
category ?category=Wildkalb Filtert nach spezifischer Wildart
from ?from=2024-01-01 Exportiert ab Datum (YYYY-MM-DD)
to ?to=2024-12-31 Exportiert bis Datum (YYYY-MM-DD)
Beispiel URLs
Export für Rotwild:
{{ request.url_root }}export_csv?category=Rotwild
Export mit Datumsbereich:
{{ request.url_root }}export_csv?from=2024-01-01&to=2024-12-31
Export für Damwild im Jahr 2024:
{{ request.url_root }}export_csv?category=Damwild&from=2024-01-01&to=2024-12-31
Hinweise:
  • URLs können direkt in Browser eingegeben oder per Script aufgerufen werden
  • Export ist öffentlich zugänglich - keine Anmeldung erforderlich
  • Dateiname wird automatisch aus Einstellungen und Filtern generiert
  • CSV-Format: Standard kommagetrennt, UTF-8 kodiert
